## ğŸ¯ Common Scenarios

### Scenario 1: Adding a New Page

**Template:**
```tsx
export default function NewPage() {
  return (
    <>
      <Navbar />
      
      {/* Container with responsive padding */}
      <div className="container-custom py-4 sm:py-6 md:py-8">
        
        {/* Header */}
        <h1 className="text-2xl sm:text-3xl md:text-4xl font-bold mb-4 sm:mb-6">
          Page Title
        </h1>
        
        {/* Content Grid */}
        <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4 sm:gap-6">
          {/* Your content */}
        </div>
        
        {/* Action Button */}
        <button className="btn btn-primary w-full sm:w-auto mt-6 touch-manipulation">
          Action
        </button>
      </div>
    </>
  );
}
```

### Scenario 2: Adding a Modal

**Template:**
```tsx
<div className="fixed inset-0 z-50 flex items-start sm:items-center p-0 sm:p-4">
  {/* Backdrop */}
  <div className="absolute inset-0 bg-black/50" onClick={onClose} />
  
  {/* Modal */}
  <div className="relative bg-white sm:rounded-lg w-full max-w-2xl max-h-screen overflow-y-auto">
    {/* Close button */}
    <button 
      onClick={onClose}
      className="sticky top-2 right-2 sm:absolute sm:top-4 sm:right-4 p-2 bg-white rounded-full shadow-lg touch-manipulation"
    >
      <X className="w-5 h-5 sm:w-6 sm:h-6" />
    </button>
    
    {/* Content with responsive padding */}
    <div className="p-4 sm:p-6 md:p-8">
      {/* Your modal content */}
    </div>
  </div>
</div>
```

### Scenario 3: Adding a Form

**Template:**
```tsx
<form className="space-y-4 sm:space-y-6">
  {/* Single column on mobile, 2 on desktop */}

    <div>
      <label className="block text-sm font-medium mb-2">
        Field Name
      </label>
      <input
        type="text"
        className="input text-base"  {/* 16px for iOS */}

    </div>
    
  </div>
```

## ğŸ“± Mobile-First Checklist

When creating new components, verify:

- [ ] Text readable on 375px screen?
- [ ] Buttons minimum 44x44px?
- [ ] Grid adapts to screen size?
- [ ] No horizontal scroll?
- [ ] Touch-friendly spacing?
- [ ] Images responsive?
- [ ] Forms don't trigger zoom?
- [ ] Navigation accessible?

## ğŸ“ Pro Tips

### Tip 1: Start Small
Always design for mobile first (375px), then scale up.

### Tip 2: Test Often
Test on mobile after every major change.

### Tip 3: Use Utility Classes
Tailwind's responsive classes make it easy:
```tsx
className="text-sm sm:text-base md:text-lg"
```

### Tip 4: Think Touch
Mobile users use thumbs - make targets big!

### Tip 5: Embrace Stacking
It's okay to stack vertically on mobile - better than cramming.
